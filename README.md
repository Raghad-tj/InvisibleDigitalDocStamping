# Invisible Digital Document Stamping
# 🔒 Final Project Submission: أداة الختم والتحقق الآمن للملفات (Python Sealing & Verification)

## I. نظرة عامة على المشروع (Project Overview)

-   **الهدف:** تنفيذ آلية رقمية للختم (التوقيع الآمن) والتحقق لضمان
    سلامة محتوى الملفات وعدم التلاعب بها. يتم توليد الختم باستخدام
    خوارزمية HMAC-SHA256.
-   **التطبيقات:**
    -   **الصور (PNG/JPG/JPEG):** يتم توليد بصمة بناءً على بايتات
        البيكسلات وتخزينها في البيانات الوصفية لملف PNG.
    -   **ملفات PDF:** يتم توليد بصمة بناءً على المحتوى النصي الكامل
        المستخرج من المستند وتخزينها في metadata.
-   **النتائج:** تتيح الأداة للمستخدم التحقق من أن الملف المختوم لم
    يتعرض لأي تعديل بعد عملية الختم الأصلية.

## IV. تعليمات التشغيل وإعادة الإنتاج (Run Instructions)

يتضمن المشروع خليتين رئيسيتين:

### 1. خلية الختم (Sealing)

**الوصف:** رفع ملف → توليد الختم → حفظ نسخة مختومة وتنزيلها.

  الوظيفة    الوصف
  ---------- ----------------------------------
  المدخلات   رفع صورة أو PDF
  العملية    حساب بصمة وتخزينها داخل metadata
  المخرجات   إنشاء ملف باسم يبدأ بـ `sealed_`

------------------------------------------------------------------------

### 2. خلية التحقق (Verification)

**الوصف:** رفع ملف مختوم → استخراج البصمة → إعادة حساب البصمة →
المقارنة.

  النتيجة                    التفسير
  -------------------------- ------------------------------
  ✅ **الملف أصلي**          لم يحدث أي تغيير على المحتوى
  ❌ ** لايوجد ختم امني بالملف**   المحتوى أو الختم غير مطابق
